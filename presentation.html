The Problem at DevX Ltd

Manual, fragmented & error-prone

Attendance lived in a paper sign-in book, leave in email threads, and payroll in hand-typed Excel — none of it connected.

🪪

Proxy attendance

A paper book can't prove the person signing in is actually present — buddy-punching went undetected.

📨

Lost leave requests

Approvals took days, balances were calculated by hand, and disputes were common.

🧮

Payroll errors

Manual Excel with PAYE, RSSB & loan deductions was slow and easy to get wrong.

🔌

No synchronization

An employee on approved leave still showed up as absent — records constantly disagreed.

👁️

Zero visibility

Managers planned on spreadsheets that were already out of date.

📋

No audit trail

No record of who changed what, or who approved which request.

The Signature Feature

Three-layer anti-proxy check-in

Every check-in must pass all three independent layers. A weakness in one cannot create a false record.

1
📍

GPS Geofence

Browser location is checked against DevX HQ coordinates & radius.

→
2
🔲

Rotating QR

A time-limited token validated server-side — can't be screenshotted & reused.

→
3
🙂

Face Verify

face-api.js compares a 128-d descriptor in-browser. No image ever stored.

✓ All three pass → verified record created → Sync Engine triggered

Internal Team Chat

Conversations next to the work

A full messaging module built in — no Slack needed. Work talk sits beside attendance, leave & task data, with everything logged to the audit trail.

📢

Three channel types

Public channels for all staff, team channels tied to a manager's group, and 1-to-1 direct messages created on first message.

📎

Rich messages

Send text with file attachments — images, video, documents & audio — react with emoji, and see read receipts.

⏳

Ephemeral status

Post a short-lived status update that expires after a set time, so teammates know what you're on.

🔔

Unread counts

Live unread badges per channel keep nothing missed, refreshed by polling.

🗑️

Soft delete

Messages can be removed, leaving a clean deletion notice — nothing silently vanishes.

🛡️

Membership-gated

You can only post to channels you've joined — enforced server-side & written to the audit log.

Security & Privacy by Design

Sensitive data, handled with care

🔑

Hashed passwords

bcrypt via PHP password_hash — never stored in plain text.

🎫

JWT + revocable sessions

Every API request carries a token, validated before any logic runs.

💉

SQL-injection safe

All DB access via PDO prepared statements; output escaped.

🧬

Encrypted face data

AES-encrypted 128-d descriptor only — no facial images ever kept.

⏱️

GPS auto-purge

Presence-monitoring coordinates purged after 24 hours.

🚧

Role gate

Permission check at the top of each backend file blocks cross-role access.
